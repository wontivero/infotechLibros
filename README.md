# InfoTech - Gestión de Impresiones Escolares

SPA (Single Page Application) diseñada para optimizar la gestión de pedidos de libros escolares y la atención al cliente por WhatsApp.

## Características
- **Dashboard Kanban:** Gestión visual de estados de pedidos (Pendiente, Imprimiendo, Terminado, Entregado).
- **Integración IA (Gemini):** Extracción automática de datos de pedidos desde texto de WhatsApp.
- **Catálogo:** ABM de libros con cálculo automático de precios y gestión de imágenes.
- **CRM Waitlist:** Gestión de lista de espera para libros sin stock.

## Tecnologías
- **Frontend:** HTML5, CSS3 (Bootstrap 5), JavaScript Vanilla (ES Modules).
- **Backend:** Firebase (Firestore & Storage).
- **IA:** Google Gemini API.

## Instalación
1. Clonar el repositorio.
2. Abrir `index.html` usando un servidor local (ej: Live Server de VS Code) para evitar errores de CORS con los módulos JS.
